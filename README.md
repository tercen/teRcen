# Tercen R client

## Install teRcen package
 
```R
install.packages('remotes')

remotes::install_github("tercen/teRcenApi", ref = "0.12.2", args="--no-multiarch")
remotes::install_github("tercen/teRcen", ref = "0.12.6", args="--no-multiarch")
remotes::install_github("tercen/teRcenApi", args="--no-multiarch")
remotes::install_github("tercen/teRcen", args="--no-multiarch")
```
